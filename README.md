# 🤖 Jarvis - Python Voice Assistant

Jarvis ek simple Python-based voice assistant hai jo speech recognition aur text-to-speech ka use karke user ke commands follow karta hai.

Is project ka aim hai Python me voice automation aur AI assistant ka basic idea samajhna.

__

## 🚀 Features

- 🎙️ Voice command recognition  
- 🔊 Text-to-Speech (pyttsx3)  
- 🌐 Open websites (Google, YouTube, etc.)  
- 🎵 Play music  
- 😂 Tell jokes  
- 🕒 Tell current time & date  
- 🧮 Do simple calculations  
- 🖥️ Shutdown & Restart system  
- 🔁 Infinite listening mode ("Hello Jarvis" wake word)

---

## 🛠️ Technologies Used

- Python  
- SpeechRecognition  
- pyttsx3  
- PyAudio  
- simpleeval  
- webbrowser  
- os  
- datetime
